MobMenu

We struggled through a lot of projects with different ways of creating mobile menus that works as we wanted. We decided to create a way of doing it, this is it.

This is an example, of course you can add an extra menu to the right with the same logic. And a tip if you want to use a fixed header in mobile is to locate the mobile-header in a relative positioned container. Since the fixed header stoops working in Windows mobile if it's located in a fixed container.
